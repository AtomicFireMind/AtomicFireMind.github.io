Hexo启航:+GitHub Pages部署

发表于 2021-05-22 更新于 2021-06-05 分类于 更多 ， 静态博客 ， Hexo

\qquad继本地博客搭建完成，现在让它能够实现远程共享。没有进行调研，直接地就选择了GitHub Page，因为Hexo提供了其一键部署的功能，非常方便简单。

\qquad说其简单，主要是Hexo部署步骤很少：

GitHub仓库创建及插件安装

\qquad仓库创建就不多说了，自行百度/谷歌吧，不过因为部署博客，仓库创建有点小细节需要注意:
\qquad\qquad - 所部署的仓库必须是Public,否则网页将不能访问
\qquad\qquad - 仓库名称(repository name)需要是：用户名.github.io \\ \qquad\qquad#用户名即github账户名称，比如：visioner3d.github.io
\qquad\qquad - 仓库branches设置需要与_config.yml配置一致
\qquad部署插件安装就一句话，进入博客顶目录，执行:npm install hexo-deployer-git --save
